# Open-source: developer CLI tooling

- **Role / Employer:** Personal / open source
- **When:** 2022-05 to present
- **Context:** Internal developers lacked a fast, scriptable way to inspect service
  dependencies; existing tools were slow and GUI-only.
- **Action:** Built and open-sourced `depgraph`, a Go CLI that renders service dependency
  graphs from telemetry. Maintain it, triage issues, and review community PRs.
- **Metric:** 2,300+ GitHub stars; adopted by 40+ teams (per README "used by" + issues);
  reduced dependency-audit time from hours to minutes for adopters.
- **Skills:** Go, CLI/tooling, open source, developer experience, graph algorithms
- **Source:** Public GitHub repo (stars, contributors, issue history) — verifiable URL.
