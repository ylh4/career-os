# Cloud cost reduction program

- **Role / Employer:** Staff Software Engineer @ Nimbus Retail
- **When:** 2024-09 to 2025-03
- **Context:** Cloud spend was growing faster than revenue; finance flagged compute as the
  top line item with no clear owner.
- **Action:** Started a cross-team efficiency effort — built per-service cost attribution
  dashboards, right-sized over-provisioned fleets, moved batch workloads to spot capacity,
  and set autoscaling SLOs. Drove adoption through a monthly efficiency review I chaired.
- **Metric:** Reduced annual cloud spend by **$1.8M/yr (-23%)** with no SLA regressions;
  cost-per-order down 31%; created the dashboards still used for ongoing governance.
- **Skills:** cloud cost optimization, autoscaling, FinOps, AWS, platform, influence
- **Source:** Finance savings report FY25; efficiency review deck series; manager attestation.
